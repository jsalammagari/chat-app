# Chat-app

**1. Angular-Springboot-Stomp**

> Run springboot-server
```
./gradlew bootRun
```

> Run angular-springboot-chatapp
```
ng serve --port 4201
```

> Open browser
```
http://localhost:4201
```
**2. ReactJs-NodeJs-Socket.io**

> Run nodejs-server
```
node server.js 
```

> Run react-nodejs-chatapp
```
npm start
```

> Open browser
```
http://localhost:3000
